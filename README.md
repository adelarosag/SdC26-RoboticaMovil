# SdC26-RoboticaMovil

Repositorio con recursos del taller de Robótica Móvil de la Semana de Ciencia 2026 en IIT-UACJ, Ciudad Juárez, Chihuahua, México..

En estos archivos encontrarás las presentaciones usadas, junto con códigos de ejemplo. 

NOTA: El taller está diseñado para introducir conceptos fundamentales de la robótica móvil, además de generar interés en estudiantes de nivel medio-superior. Presupone conocimiento básico o mínimo de programación. 

## Contenido del taller

## Día 1 

- Introducción a la robótica
- ¿Qué es la robótica móvil?
- Historia breve de la robótica
- Arquitectura básica de un robot
  - Sensores
  - Procesamiento
  - Actuadores
- Tipos de robots móviles
  - Robots con ruedas
  - Robots con patas
  - Robots aéreos y submarinos
- Introducción al robot Zumo

- Sensores del robot Zumo
  - Sensores infrarrojos
  - Sensores de proximidad
  - IMU (acelerómetro y giroscopio)
- Actuadores y locomoción
  - Motores DC
  - Control mediante puente H
  - Movimiento diferencial
- Introducción a Arduino
  - Arquitectura básica
  - Entradas y salidas digitales
  - PWM
- Primeros programas para el robot Zumo

## Día 2

- Repaso de conceptos del día anterior
  - Sensores y actuadores
  - Control de motores
  - Lectura de sensores

- Introducción al control en robótica
  - Sistemas de control
  - Control en lazo cerrado

- Concepto de error
  - Diferencia entre objetivo y estado actual
  - Corrección del movimiento del robot

- Introducción al control PID
  - Control proporcional (P)
  - Control integral (I)
  - Control derivativo (D)

- Aplicación del control en robótica móvil

- Robots seguidores de línea
  - Funcionamiento de sensores de línea
  - Detección de contraste (blanco y negro)
  - Estrategias de seguimiento

- Implementación básica de un seguidor de línea
  - Lectura de sensores
  - Corrección de trayectoria
  - Ajuste de parámetros

- Introducción a robots sumo
  - Reglas básicas de la competencia
  - Arena de combate
  - Estrategias de ataque y defensa

- Análisis de competencias de robots sumo
- Programación del robot Pololu Zumo para competencia

## Material

Para este taller se utilizarán las siguientes plataformas:

- Arduino
    - Software [Arduino IDE (1.8.9 +)](https://docs.arduino.cc/software/ide/)
    - Microcontrolador Arduino UNO Rev 3
- [Pololu Zumo Robot for Arduino (Assembled with 75:1 HP Motors)](https://www.pololu.com/product/2510)
    - Paquetería para Arduino 'zumo-shield-arduino-library', disponible en el IDE o en GitHub: https://github.com/pololu/zumo-shield-arduino-library
